Creatable Dropdown

Dropdown with inline create capability using <lyte-dropdown> Slyte DOM. Type to search existing options or create new ones on the fly with a "+ Create" action row.

Form Figma Synced 4 Variants × 1 Size
Default — Empty / Placeholder
Base
HTML
<lyte-dropdown lt-prop-placeholder="Search or create..."
               lt-prop-search="true"
               class="zc-dropdown zc-creatable-dropdown">
  <template is="registerYield" yield-name="yield">
    <lyte-drop-box><lyte-drop-body>
      <lyte-drop-item data-value="opt1">Option 1</lyte-drop-item>
      <lyte-drop-item data-value="opt2">Option 2</lyte-drop-item>
      <lyte-drop-item data-value="opt3">Option 3</lyte-drop-item>
    </lyte-drop-body></lyte-drop-box>
  </template>
</lyte-dropdown>

}

With Create Row
Create "New Item"
No Match + Create
No matching options
Create "Custom Value"
HTML
<!-- Creatable Dropdown with "+ Create" action row -->
<lyte-dropdown lt-prop-placeholder="Search or create..."
               lt-prop-search="true"
               class="zc-dropdown zc-creatable-dropdown">
  <template is="registerYield" yield-name="yield">
    <lyte-drop-box><lyte-drop-body>
      <lyte-drop-item data-value="opt1">Option 1</lyte-drop-item>
      <lyte-drop-item data-value="opt2">Option 2</lyte-drop-item>
      <lyte-drop-item data-value="opt3">Option 3</lyte-drop-item>
      <!-- Create row appended when typed value has no match -->
      <lyte-drop-item class="zc-creatable-dropdown-create-row"
                       data-value="__create__">
        <i class="fas fa-plus"></i>
        Create "New Item"
      </lyte-drop-item>
    </lyte-drop-body></lyte-drop-box>
  </template>
</lyte-dropdown>

<!-- No match — only create row shown -->
