Toggle

Toggle switch using <lyte-checkbox> switch type Slyte DOM.

Form Figma Synced 2 States × 3 Sizes
Default — Off
Off (Unchecked)
Off with Label
HTML
<lyte-checkbox lt-prop-type="switch" class="zc-toggle">
  <template is="registerYield" yield-name="yield">
    <span>Toggle label</span>
  </template>
</lyte-checkbox>
SCSS
.zc-toggle {
  /* Default off state — neutral track */
}
.zc-toggle .lyteCheckboxBox {
  background: var(--zc-color-neutral-surface-neutral-surface-default, #D5D6D9);
  border-radius: 999px;
  position: relative;
  transition: background 0.2s ease;
}
.zc-toggle .on-btn {
  background: #FFFFFF;
  border-radius: 50%;
  transition: transform 0.2s ease;
}

With Label — Off & On
Off + Label
On + Label
Multiple Toggles
HTML
<!-- Off with label -->
<lyte-checkbox lt-prop-type="switch" class="zc-toggle">
  <template is="registerYield" yield-name="yield">
    <span>Enable notifications</span>
  </template>
</lyte-checkbox>

<!-- On with label -->
<lyte-checkbox lt-prop-type="switch" lt-prop-checked="true" class="zc-toggle">
  <template is="registerYield" yield-name="yield">
    <span>Notifications enabled</span>
  </template>
</lyte-checkbox>

<!-- Multiple toggles in a list -->
<div style="display:flex;flex-direction:column;gap:12px;">
  <lyte-checkbox lt-prop-type="switch" lt-prop-checked="true" class="zc-toggle">
    <template is="registerYield" yield-name="yield">
      <span>Email alerts</span>
    </template>
  </lyte-checkbox>
  <lyte-checkbox lt-prop-type="switch" class="zc-toggle">
    <template is="registerYield" yield-name="yield">
      <span>SMS alerts</span>
    </template>
  </lyte-checkbox>
</div>
