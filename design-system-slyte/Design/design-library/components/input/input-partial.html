Text Input

Text input field using <lyte-input> Slyte DOM. States: Empty, Filled, Focus, Hover, Disabled, Read-only, Error, Success with optional prefix/suffix icons and label support.

Form Figma Synced 10 States × 1 Size
Default — Empty Input
Base
HTML
<!-- Lyte Input — Default empty -->
<lyte-input lt-prop-type="text" lt-prop-placeholder="Enter text..." lt-prop-appearance="box"></lyte-input>
SCSS
/* lyte-input renders: .lyteInput > .lyteField > input */
lyte-input .lyteField {
  border: 1px solid var(--zc-color-neutral-border-neutral-border-default, #D5D6D9);
  border-radius: var(--zc-radius-r, 6px);
  background: var(--zc-color-black-white-surface-white-surface, #FFFFFF);
  transition: border-color 0.15s ease, box-shadow 0.15s ease;
}
lyte-input .lyteField input {
  font-family: 'ZohoPuvi', sans-serif;
  font-size: var(--zc-p2-font-size, 14px);
  color: var(--zc-color-neutral-text-neutral-text-default, #37383C);
}
lyte-input .lyteField input::placeholder {
  color: var(--zc-color-neutral-text-neutral-text-subtle, #9EA0A7);
}

Focus — Blue Border with Shadow
Base (click to focus)
HTML

/* lyte-input adds .lyteInputFocus on focus */
lyte-input.lyteInputFocus .lyteField,
lyte-input .lyteField:focus-within {

  box-shadow: 0px 2px 2px 0px rgba(13, 78, 242, 0.05), 0px 2px 4px 0px rgba(13, 78, 242, 0.08);
