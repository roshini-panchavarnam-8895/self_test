Radio Card

Radio button presented as a selectable card/tile using <lyte-radiobutton> Slyte DOM with label, description, and optional icon.

Form Figma Synced 6 Variants × 2 Sizes
Default — Unselected
Default
Small

<!-- Default Radio Card -->
<div class="zc-radio-card">
  <lyte-radiobutton name="demo-group">
    <template is="registerYield" yield-name="yield">
      <span class="zc-radio-card-label">Option A</span>
    </template>
  </lyte-radiobutton>
</div>

<!-- Small Radio Card -->
