Tag Types

Tag/chip UI elements for labels, categories, and filters. Uses custom .zc-tag HTML elements.

Form Figma Synced 5 Colors × 3 Sizes
Default Tags — All Sizes
Small Label Success Warning Error Neutral
Medium Label Success Warning Error Neutral
Large Label Success Warning Error Neutral
HTML
<!-- Default Tags -->
<span class="zc-tag zc-tag-primary zc-tag-md">Label</span>
<span class="zc-tag zc-tag-success zc-tag-md">Success</span>
<span class="zc-tag zc-tag-warning zc-tag-md">Warning</span>
<span class="zc-tag zc-tag-error zc-tag-md">Error</span>
<span class="zc-tag zc-tag-neutral zc-tag-md">Neutral</span>
