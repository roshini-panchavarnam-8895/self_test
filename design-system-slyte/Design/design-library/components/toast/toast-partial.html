Toast

Toast notification component using <lyte-banner> Slyte DOM with custom .zc-toast classes via lt-prop-class. Supports Info, Success, Warning, and Error types with close button, action button, and positioned containers.

Floating Figma Synced 4 Types × 6 Positions
Info Toast
Default
Long Text
HTML
<lyte-banner lt-prop-floating="false" lt-prop-show="true"
  lt-prop-show-close-button="false"
  lt-prop-class="zc-toast zc-toast-info">
  <template is="registerYield" yield-name="yield">
    <span class="zc-toast-icon"><i class="fas fa-info-circle"></i></span>
    <span class="zc-toast-message">Info toast message.</span>
    <button class="zc-toast-close">&times;</button>
  </template>
</lyte-banner>
SCSS
.zc-toast-info {
    background: var(--zc-color-info-surface-info-surface-subtle, #F5F8FE);
    border-left-color: var(--zc-color-info-border-info-border-default, #0D4EF2);
    color: var(--zc-color-info-text-info-text-bold, #051F61);
}

    position: fixed;
    z-index: 9999;
    display: flex;
    flex-direction: column;
