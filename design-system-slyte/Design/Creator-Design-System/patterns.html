Top Bar — Pattern

Source: Figma Top Bar (9413:361). Full app navigation header for Creator. 48px tall, deep navy bg (--primary-surface-bold). Layout: Logo · Workspaces dropdown · Spacer · Right group (apps grid · separator · avatar).

Default state

Creator

Anatomy

  • Logo — product wordmark + diamond icon, white
  • Workspace dropdown — pill button with 1px primary-minimal border, 32px height, transparent until hover
  • Spacer — flex grow that pushes right group to the end
  • Apps grid icon — 3×3 dot grid (waffle), opens Zoho apps overlay
  • Separator — 1px vertical line, 20% white
  • Avatar — 32px circle with user initials
